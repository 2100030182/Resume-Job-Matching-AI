# Resume–Job Matching AI with Skill Gap Intelligence

## Overview
This project builds an AI-powered system that analyzes resumes and compares them with job descriptions to calculate compatibility scores.

## Features
- Resume parsing from PDF
- Job description analysis
- Skill extraction using NLP
- Resume-job similarity scoring
- Skill gap detection
- Personalized recommendations
- Interactive dashboard

## Tech Stack
- Python
- NLP (spaCy, NLTK)
- Machine Learning (scikit-learn)
- Streamlit
- GitHub
